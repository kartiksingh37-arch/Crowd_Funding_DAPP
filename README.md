# Crowdfunding DApp

A decentralized crowdfunding platform built with Solidity, React, Ethers.js and Hardhat.

## Features

- Connect Wallet (MetaMask)
- Create Campaign
- Donate to Campaign
- Campaign Details Page
- Progress Tracking
- Donator List
- Withdraw Funds
- Cancel Campaign

## Tech Stack

- Solidity
- Hardhat
- React
- Ethers.js
- Tailwind CSS
- MetaMask

## Smart Contract Functions

- createCampaign()
- donateToCampaign()
- getCampaigns()
- getDonators()
- withdrawFunds()
- cancelCampaign()
